--------------------------------DESCRIPTION------------------------------------ 
This project optimizes the Blur Filter by parallelizing it with CUDA.
-------------------------------------------------------------------------------


--------------------------------COMPILE AND RUN-------------------------------- 
Compile using the Makefile:
	make

To run the code:
	$ ./blur_filter {size}

For example:
	$ ./blur_filter 4096

To clean up executable files:
	make clean
